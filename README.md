# 港町の読書部屋 v4

GitHub Pages公開用フォルダです。

## 公開ファイル

- `index.html`: GitHub Pagesで最初に表示されるファイル
- `reading_room_app_v4.html`: 元ファイル名を残したコピー

## GitHub Pagesで公開する方法

1. このフォルダの中身をGitHubリポジトリにアップロードします。
2. GitHubのリポジトリ画面で `Settings` を開きます。
3. `Pages` を開きます。
4. `Build and deployment` の `Source` を `Deploy from a branch` にします。
5. `Branch` を `main`、フォルダを `/ (root)` にします。
6. `Save` を押します。

公開URLは数十秒から数分後に表示されます。

## iPhoneでバーコード読み取りを使う場合

GitHub PagesのURLはHTTPSなので、iPhone Safariでカメラ読み取りを試せます。
`file://` で直接開いた場合は、ブラウザ制限によりカメラが使えないことがあります。
